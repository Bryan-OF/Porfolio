PartPicker Recreated website

The assignment was to revamp an existing Web site, but with an emphasis on using semantic HTML and CSS. The challenge was to format each HTML element directly in CSS, without using classes. This approach gave me a unique opportunity to learn HTML and CSS in a hands-on way and better understand how these technologies work together to create a visually appealing and functional Web site. Despite the limitations of not being able to use classes, I found the assignment both fun and educational, and gained valuable skills that will come in handy for future projects.

You can take a live look at my prototy site here: www.pcpartpicker.com
