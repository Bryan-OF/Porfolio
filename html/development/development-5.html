PartPicker Recreated website

For this project, I recreated a website called PC Part Picker. This Website helps users put together their own PC by choosing and combining components. The assignment was to revamp an existing Web site, but with an emphasis on using semantic HTML and CSS.

Aim of the project

The challenge was to format each HTML element directly in CSS, without using classes. This approach gave me a unique opportunity to learn HTML and CSS in a hands-on way and better understand how these technologies work together to create a visually appealing and functional Web site. Despite the limitations of not being able to use classes, I found the assignment both fun and educational, and gained valuable skills that will come in handy for future projects.

Personal changes

What makes my re-created website different from the original one is the following:

• A more responsive design that works on both desktops and mobile devices.

• A clear and easy-to-read text, even for colour-blind people.

• Effectively separating the ads from main content.

• A dark mode option that is easily accessible.

• Improved user interaction with hover effects and clear call-to-action buttons.

• An interactive carousel for build guides.

Final conclusions

I am happy to say that my rebuilt version of PC Part Picker is a bit more responsive and accessible than the original website. An improvement for the future would be to make all elements fully responsive and more inclusive for visually impaired users. And improve the accessibility features such as screen reader support and keyboard navigation.
