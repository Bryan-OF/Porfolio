Journey Inside Myself

For my “Playing with CSS” project, I recreated one of my CRTL CARE Illustrations in HTML/CSS and animated it with CSS Animation Techniques. We were instructed to complete an animated coding project with an interactive function in the span of one week, with the condition being that it must contain only code. I decided to practice with a manual, using an automatic Light-on-dark color scheme.

Project Goal

For this project, we had to choose a characteristic image or artwork and recreate it with HTML and CSS. I aimed to improve and deepen my CSS skills by experimenting with different techniques and features. This project was suitable for students who have discovered the joy of coding and wanted guidance to explore more of what CSS has to offer. This project was ideal for me because I was also considering a career in development to create more impressive prototypes.

Things I learned and applied

• Selectors and pseudo-elements: To target specific elements without using any ID Attribute.

• Fonts: How to pull and implement different fonts so that they're organized in your code.

• (Transparent) colors: Mixing colors and transparency to add depth and layering.

• (Multiple) backgrounds, (repeating) gradients, and (text) shadows: How to fully utilize multiple backgrounds, repeating gradients, and text shadows to enhance visual effects.

• Borders and outlines: Learn creative and different ways to use them.

• (3D) transforms: Give elements specific animations for a 3D look.

• Transitions and animations: Adding transitions and animations for interaction and movement.

• Positioning with flex, grid, relative, and absolute: Improve different positioning techniques for layout and structure.

• Shape-outside and clip-path: Using CSS Tricks to create unique shapes, layouts, and effects.

• Variables: Applying CSS variables for reusable and easily adjustable styles.

Final Outcome

By the end of this project, I improved my CSS techniques and skills. It gave me a better understanding of how to work more effectively and efficiently on projects with short deadlines. After this course, I wanted to learn more about animation and how to improve my understanding of it.
